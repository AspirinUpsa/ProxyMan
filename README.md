# ProxyMan

**Прокси Менеджер** — утилита для быстрого управления системным прокси в Windows через системный трей.

## 🚀 Возможности

- **Быстрое переключение прокси** — включение/отключение одним кликом левой кнопкой мыши по иконке в трее
- **Настройка параметров прокси-сервера** — указание адреса, порта, исключений
- **Управление исключениями** — список адресов, для которых прокси не используется
- **Автозагрузка с Windows** — автоматический запуск при старте системы
- **Работа в системном трее** — минималистичный интерфейс без лишних окон
- **Аутентификация** — поддержка имени пользователя и пароля для прокси-сервера
- **Обход локальных адресов** — автоматическое исключение локальных ресурсов из прокси
- **Сохранение состояния** — запоминание последнего состояния прокси между запусками
- **Единственный экземпляр** — защита от запуска нескольких копий приложения

## 📋 Системные требования

- **ОС:** Windows (используется WinInet API)
- **Фреймворк:** Qt 5.x / Qt 6.x
- **Компилятор:** MinGW или MSVC

## 💡 Использование

### Первый запуск

1. Запустите `ProxyMan.exe`
2. Иконка появится в системном трее
3. Нажмите правой кнопкой мыши для открытия контекстного меню

### Настройка прокси

1. **ПКМ на иконке** → «Настройки прокси»
2. Укажите параметры:
   - **Адрес сервера** — IP или домен прокси (например, `192.168.1.1` или `proxy.company.com`)
   - **Порт** — порт прокси-сервера (по умолчанию `8080`)
   - **Исключения** — адреса через точку с запятой (например, `localhost;127.0.0.1;*.local`)
   - **Аутентификация** — логин и пароль (если требуется)
3. Нажмите «Сохранить»

### Переключение прокси

- **ЛКМ на иконке** — быстрое переключение между режимами:
  - 🟢 **Прокси включён** — подключение через настроенный прокси-сервер
  - 🔴 **Прямое подключение** — работа без прокси

### Контекстное меню (ПКМ)

- **Переключить прокси** — включение/отключение
- **Настройки прокси** — открытие диалога настроек
- **Информация** — текущее состояние и параметры
- **Выход** — завершение работы приложения

### Автозагрузка

В диалоге настроек доступна опция «Запускать программу при старте Windows». При включении приложение добавляется в реестр:
```
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run
```

## 🗂️ Хранение настроек

Настройки сохраняются в реестре Windows:

```
HKEY_CURRENT_USER\Software\ProxyMan
```

Параметры:
- `ProxyServer` — адрес прокси-сервера
- `ProxyPort` — порт
- `ProxyExceptions` — список исключений
- `ProxyBypassLocal` — обход локальных адресов
- `ProxyUser` — имя пользователя
- `ProxyPassword` — пароль
- `LastProxyState` — последнее состояние (вкл/выкл)

## 🛠️ Технические детали

### Работа с WinInet API

Приложение использует функции Windows WinInet для управления системными настройками прокси:
- `InternetSetOption()` — установка параметров подключения
- `InternetQueryOption()` — получение текущих настроек

### Защита от множественных экземпляров

Используется комбинация `QSharedMemory` и `QSystemSemaphore` для предотвращения запуска нескольких копий приложения.
